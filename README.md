# Portafolio - Diseñador Freelancer

Sitio web profesional de portafolio para mostrar servicios de diseño y desarrollo web.

## 🎨 Características

- Diseño moderno y responsivo
- Navegación intuitiva
- Sección de servicios (Diseño Web, Aplicaciones Web, E-Commerce)
- Formulario de contacto funcional
- Optimizado para móviles y escritorio

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Diseño responsivo con Grid y Flexbox
- **JavaScript** - Interactividad (próximamente)
- **Normalize.css** - Normalización de estilos
- **Google Fonts** - Tipografía Krub
